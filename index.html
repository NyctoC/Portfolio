Bring-U

Bring-U is an independent web project built with Django, designed to enhance the student experience on campus. It provides a platform for food deliveries from local restaurants and carpooling services. My primary role in this project was developing a significant portion of the user interface and a AI capable of recognizing text from images. This AI would automatically format and generate complete menus from a single photo, making it easier for restaurants and customers to create menus or list services without manually entering each item.

Gravity

An online clothing store built with Django, known for its modern design, strong visual appeal, and AI-powered outfit recommendations. The AI suggests outfits based on a simple prompt entered by the customer, tailoring suggestions to the occasion. My role in this project involved developing the entire front-end and programming the AI system, which operates using the OpenAI API.

SaSa

My role in this project involved developing a significant portion of the front-end and handling communication between the database model and the website.
